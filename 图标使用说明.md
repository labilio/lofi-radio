# 🎨 图标使用说明

## 📋 当前状态

已生成 SVG 图标文件：`icon.svg`

## 🔄 转换为 Windows 图标（.ico）

### 方法1：在线转换（推荐，最简单）

1. 访问在线转换工具：
   - https://convertio.co/zh/svg-ico/ （推荐）
   - https://www.icoconverter.com/
   - https://cloudconvert.com/svg-to-ico

2. 上传 `icon.svg` 文件

3. 下载转换后的 `icon.ico` 文件

4. 将 `icon.ico` 文件放到项目根目录（与 `main.js` 同级）

### 方法2：使用 ImageMagick（如果已安装）

```bash
magick convert icon.svg -resize 256x256 icon.ico
```

### 方法3：使用在线图标生成器

1. 访问 https://www.favicon-generator.org/
2. 上传 `icon.svg`
3. 下载生成的 `favicon.ico`
4. 重命名为 `icon.ico` 并放到项目根目录

## ✅ 验证

转换完成后，确保项目根目录中有 `icon.ico` 文件：

```
lofi-radio/
├── icon.ico          ← 这个文件必须存在
├── icon.svg
├── main.js
├── package.json
└── ...
```

## 🚀 使用

图标文件准备好后：

1. **任务栏图标**：应用启动后，任务栏会显示你的自定义图标
2. **系统托盘图标**：右下角托盘区域会显示你的自定义图标

## 📝 注意事项

- Windows 优先使用 `.ico` 格式
- 如果没有 `.ico` 文件，会自动尝试使用 `.png` 文件
- 图标文件应该放在项目根目录
- 建议图标尺寸：256x256 像素（.ico 格式支持多尺寸）

## 🎯 图标设计

当前图标设计：
- **背景**：靛蓝色圆形 (#6366f1)
- **符号**：白色音乐符号 (♪)
- **风格**：简洁现代，适合深色和浅色主题

如果需要自定义图标，可以：
1. 编辑 `icon.svg` 文件
2. 重新转换为 `.ico` 格式
3. 替换 `icon.ico` 文件
